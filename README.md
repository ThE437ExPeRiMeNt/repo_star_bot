# Repo Star Bot

This script automates the process of creating Google accounts, then using those to create GitHub accounts, and finally starring a specified GitHub repository.

**Disclaimer:** Use this script responsibly and be aware of the terms of service of Google and GitHub.  Automated account creation may violate their terms.

**Dependencies:**
*   Python 3.6+
*   Selenium
*   undetected-chromedriver
*   Other libraries listed in `requirements.txt`

**Setup:**

1.  Install dependencies: `pip install -r requirements.txt`
2.  Configure settings in `config.py` (e.g., target repository).
3.  Run the script: `python main.py`
